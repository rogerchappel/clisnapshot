# Roadmap

## Now

- Ship a small local-first CLI snapshot MVP.
- Keep snapshots deterministic, readable, and reviewable.
- Support config-driven cases, update mode, scrubbing, diffs, and fixture-backed tests.

## Next

- Add JSON schema for `clisnapshot.config.json`.
- Improve diff context and snapshot parsing.
- Document recipes for npm, pnpm, Python CLIs, and release smoke tests.

## Later

- Consider watch mode once the core workflow is stable.
- Consider richer reporters for CI annotations.
- Consider package-manager helpers without locking users into one framework.

## Not planned

- Hosted snapshot storage.
- Hidden telemetry.
- Mandatory accounts or cloud services.
